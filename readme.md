# My vimrc files

## Installation

Clone and move to .vim. Copy vimrc_root to ~/.vimrc

## Install Vundle
cd ~/.vim
mkdir bundle
git clone http://github.com/gmarik/vundle.git ~/.vim/bundle/vundle

## Update installed bundles

vim
:BundleInstall!

## Update Command-T

cd ~/.vim/bundle/Command-T
rake make


Enjoy!
